          Projeto Banco MoneyMaker


O projeto foi desenvolvido para área bancária no lado do cliente (ATM), que por sua vez conta com recursos típicos do segmento como: depósitos, saques e extratos. O software foi implementado com a tecnologia JAVA e a persistência no banco de dados em HIBERNATE. A arquitetura utilizada foi a MVC por ter uma maior visibilidade geral do projeto e uma organização mais efetiva. Os testes serão executados nesse ambiente de desenvolvimento que contará com técnicas tradicionais de teste, como caixa preta e caixa branca.
